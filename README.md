# Pet Care Mobile Application - FurFriend

A mobile application developed to assist pet owners in managing and enriching their daily care routines for their pets.
There are 5 primary features in this application:
1. Search: To seacrh for pet-related services nearby and pet-friendly locations.
2. Social: To upload photos and videos, sharing memorable moments with their pets while interacting with a broader pet-loving community.
3. Profile: To store important pet details.
4. Calendar: To log pet activities, schedule reminders for appointments or medication, and export the records in PDF format
5. Settings: To select preferred language.

## Tech Stack
- Java
- Firebase
